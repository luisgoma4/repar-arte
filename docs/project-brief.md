# Project Brief

## Proyecto
Web corporativa para **ReparArte**, servicio de reparación de electrónica, electricidad y automatismos en **Majadahonda, Madrid, España**.

## Objetivo
Construir una web minimalista, sencilla y sofisticada que transmita:

- confianza técnica
- atención personalizada
- servicio local de calidad
- estética limpia y tecnológica

## Prioridades
1. Claridad inmediata del servicio
2. Diseño elegante y creíble
3. Buena experiencia en móvil
4. CTA clara de contacto
5. Base lista para completar con datos reales

## Referencias de marca
- `repararte.png`
- `repararte_black.svg`

## Dirección visual
- azul profundo + azul eléctrico + cian
- fondos claros y limpios
- contraste alto
- sensación premium, no recargada
- lenguaje visual técnico pero humano

## Contenido confirmado
- nombre: ReparArte
- ubicación: Majadahonda, Madrid, España
- servicios: electrónica, electricidad, automatismos
- posicionamiento: calidad y atención personalizada

## Contenido pendiente
- teléfono
- email
- horario
- área exacta de cobertura
- detalle de servicios prioritarios

## Entrega mínima esperada
- una landing completa y funcional
- responsive
- copy base en español
- SEO local básico
- estructura fácil de ampliar
