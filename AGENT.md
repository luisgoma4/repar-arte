# AGENT.md

## Rol
Eres el agente responsable de crear de principio a fin la web de **ReparArte**, una empresa de reparación de:

- Electrónica
- Electricidad
- Automatismos

La empresa opera en **Majadahonda, Madrid, España** y se posiciona como un servicio de **calidad**, **cercano**, **profesional** y de **atención personalizada**.

Tu trabajo no es solo "montar una landing", sino construir una web **minimalista, sencilla y sofisticada**, coherente con la marca y lista para usarse como presencia comercial real.

## Objetivo principal
Diseñar e implementar una web moderna que transmita:

- Confianza técnica
- Profesionalidad
- Cercanía
- Claridad
- Servicio premium sin parecer pretencioso

La web debe ayudar a que un posible cliente entienda rápidamente:

1. Qué hace ReparArte
2. Qué tipo de problemas puede resolver
3. Dónde presta servicio
4. Por qué confiar en la empresa
5. Cómo contactar

## Contexto visual de marca
Toma como referencia `repararte.png` y `repararte_black.svg`.

### Rasgos visuales detectados
- Predominio de azules y cian eléctricos
- Identidad tecnológica y limpia
- Sensación de precisión, reparación y energía
- Logo con iconografía que mezcla electricidad, electrónica y movimiento

### Dirección estética obligatoria
- Minimalista, pero no fría
- Sofisticada, pero no corporativa genérica
- Técnica, pero accesible para clientes no expertos
- Clara, aireada y visualmente pulida

Evita una web recargada, plantillera o con apariencia de SaaS genérico.

## Paleta sugerida
Usa una base inspirada en el logo:

- Azul eléctrico
- Azul profundo
- Cian brillante
- Blanco roto o gris muy claro como fondo
- Gris antracita para texto principal

La paleta debe sentirse apropiada para una empresa de reparación técnica seria. Evita colores chillones, morados, rojos agresivos o mezclas demasiado juveniles.

## Personalidad de marca
La marca debe comunicar:

- Experiencia y precisión
- Trato directo y personalizado
- Soluciones fiables
- Profesionalidad local
- Servicio cuidado

El tono del copy debe ser:

- Cercano
- Claro
- Serio sin ser rígido
- Profesional
- Comprensible para público general

Evita lenguaje grandilocuente, promesas exageradas o frases de marketing vacías.

## Público objetivo
La web debe funcionar para clientes que buscan ayuda técnica en la zona de Majadahonda y alrededores, especialmente:

- Particulares
- Pequeños negocios
- Clientes que valoran un trato personalizado
- Personas que quieren una solución de calidad y confianza

## Alcance del proyecto
Debes encargarte de todo lo necesario para entregar una web funcional de alta calidad:

- Estructura de páginas o landing
- Arquitectura de contenidos
- Copy base en español
- Diseño visual
- Implementación frontend
- Responsive
- Estados de interacción
- Accesibilidad básica
- Preparación para SEO local básico

Si el proyecto existente ya usa una tecnología concreta, adáptate a ella. Si no existe estructura previa, elige una solución sencilla y mantenible.

## Principios de diseño
- Mucho aire y jerarquía visual clara
- Tipografía elegante y muy legible
- Secciones breves pero bien pensadas
- Uso medido de gradientes o brillos sutiles inspirados en el logo
- Componentes sobrios y con buenos acabados
- Animaciones discretas y con propósito

No hagas una web barroca. No llenes la página de bloques innecesarios.

## Estructura recomendada
Como punto de partida, prioriza una landing sólida con estas secciones:

1. Hero claro y elegante
2. Servicios principales
3. Por qué elegir ReparArte
4. Área de servicio: Majadahonda, Madrid y alrededores
5. Proceso de atención o forma de trabajo
6. Llamada a la acción de contacto
7. Footer limpio

Si ves sentido, puedes añadir una sección breve de:

- Tipos de averías o necesidades frecuentes
- Garantía de atención personalizada
- Bloque de confianza o diferenciadores

## Mensajes clave que deben aparecer
La web debe dejar claro, con naturalidad, ideas como:

- Reparación de electrónica, electricidad y automatismos
- Servicio profesional en Majadahonda
- Atención personalizada
- Soluciones de calidad
- Diagnóstico y reparación con enfoque serio y cuidado

No inventes datos concretos no confirmados, como:

- Años de experiencia
- Certificaciones
- Marcas asociadas
- Teléfonos
- Horarios
- Garantías específicas

Si faltan esos datos, deja el contenido preparado para completarlo o usa formulaciones neutras.

## Copywriting
Redacta el contenido principal en **español de España**.

### Reglas de copy
- Frases claras y directas
- Beneficio real antes que eslogan vacío
- Evita tecnicismos excesivos
- Evita texto demasiado largo
- Mantén un tono humano y profesional

## Requisitos técnicos
- La web debe verse bien en móvil y escritorio
- Debe cargar correctamente sin depender de florituras innecesarias
- Debe usar una estructura semántica limpia
- Debe tener contraste suficiente
- Debe incluir `title`, `meta description` y base de SEO local
- Debe dejar preparado el uso del logotipo de la marca

## SEO local mínimo
Incluye de forma natural referencias a:

- ReparArte
- Majadahonda
- Madrid
- reparación de electrónica
- reparación eléctrica
- automatismos

Sin forzar palabras clave de forma artificial.

## Entregable esperado
El resultado debe parecer una web real de negocio local cuidado, no una demo genérica.

Debes entregar:

- Diseño coherente con la identidad visual
- Copy base listo para revisión
- Implementación funcional
- Responsive correcto
- Código limpio y mantenible

## Criterios de calidad
Antes de dar el trabajo por terminado, comprueba:

- Que el estilo encaja con el logo
- Que la web transmite confianza y calidad
- Que el mensaje se entiende en pocos segundos
- Que la página no está sobrecargada
- Que en móvil sigue viéndose elegante
- Que hay una llamada a la acción clara

## Si falta información
Si necesitas datos concretos para cerrar el contenido, pregunta solo por lo realmente importante, por ejemplo:

- Teléfono
- Email
- Zona exacta de servicio
- Servicios concretos prioritarios
- Horarios
- Mensaje principal de marca

Mientras no exista esa información, avanza igualmente dejando la web muy bien resuelta y marcando únicamente los huecos estrictamente necesarios.
